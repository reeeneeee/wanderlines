# Custom title fonts per vibe

Each vibe uses a curated web font for its **titles** by default. To override any of
them with your own font, drop a `.ttf` (or `.otf`) here with the exact name below —
the app picks it up automatically, no code change needed. If a file isn't present,
the vibe falls back to its built-in font.

| File to add        | Vibe    | Falls back to   |
| ------------------ | ------- | --------------- |
| `retro-title.ttf`  | Retro   | Bungee          |
| `sleek-title.ttf`  | Sleek   | Space Grotesk   |
| `disco-title.ttf`  | Disco   | Monoton         |
| `earthy-title.ttf` | Earthy  | Fraunces        |
| `cute-title.ttf`   | Cute    | Baloo 2         |
| `classic-title.ttf`| Classic | Caveat          |

Tip: pick share-worthy display faces with personality — they only render on the big
titles (brand, screen headings, journey names), so body text stays readable.
